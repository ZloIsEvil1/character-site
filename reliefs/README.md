# Character relief cutouts

Put full-body transparent PNG cutouts for profile backgrounds in this folder using:

- `char1relief.png`
- `char2relief.png`
- `char3relief.png`
- `char4relief.png`
- `char5relief.png`

These files are used only on each character profile page (`character1.html` ... `character5.html`) and are shown on the right side of the profile layout.
